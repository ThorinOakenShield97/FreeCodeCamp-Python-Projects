# FreeCodeCamp-Python-Projects
Projects completed for the Python certification from freeCodeCamp.

This repository contains the five certification projects required to earn the **Python Certification** from [freeCodeCamp](https://www.freecodecamp.org/). 

Each project demonstrates a progression of foundational to advanced Python skills, ranging from fundamental data structures and Object-Oriented Programming (OOP) to complex algorithmic problem-solving and recursion.

## 🚀 Projects Included

### 1. User Configuration Manager
A program that utilizes dictionaries and tuples to create a robust configuration manager. It handles adding, updating, viewing, and deleting user settings.
* **Skills highlighted:** Dictionary manipulation, tuples, CRUD operations, and string formatting.

### 2. Budget App
An Object-Oriented project that manages different budget categories. It allows for depositing, withdrawing, and transferring funds between categories. It also includes a function that creates a bar chart showing the percentage spent in each category.
* **Skills highlighted:** Object-Oriented Programming (OOP), class methods, logic implementation, and data visualization.

### 3. Polygon Area Calculator
An OOP project that includes a `Rectangle` class and a `Square` subclass. These classes contain methods to calculate area, perimeter, diagonal, and to generate a string representation of the shape.
* **Skills highlighted:** Class inheritance, polymorphism, encapsulation, and mathematical operations.

### 4. Hash Table
An implementation of a fundamental Hash Table data structure from scratch. This project demonstrates how data is stored and retrieved efficiently under the hood, managing memory and handling potential indexing collisions.
* **Skills highlighted:** Low-level data structures, hashing algorithms, array manipulation, and collision resolution.

### 5. Tower of Hanoi Algorithm
An implementation of a recursive algorithm to solve the classic mathematical puzzle, moving disks between rods while strictly following the game's rules.
* **Skills highlighted:** Algorithm design, deep recursion, function call stacks, and logical problem solving.
